# Hardware Scope

## Wearable Hardware Objectives
- Textile-based sensor integration for non-invasive monitoring.
- Lightweight and breathable all-day wearability.
- Washable modular electronics with removable connectors.
- Reliable sensing for vitals, gait, posture, mobility, and environment.

## Required Sensor Coverage
- Cardiac and vital sensing: HR, HRV, SpO₂, respiratory, temperature, ECG-capable signal path.
- Motion sensing: walking speed, step/stride length, symmetry, balance, foot clearance.
- Posture and mobility sensing: slouch, sit-to-stand, inactivity, joint movement trends.
- Safety sensing: instability and abnormal gait signatures for fall prediction.
- Environmental sensing: heat/cold stress and exposure conditions.
- Location sensing: GPS path and geofence boundary status.

## Safety-Oriented Outputs
- Real-time instability indicators for haptic prevention.
- Emergency triggers for fall, unconsciousness, and abnormal vitals.
- Logged safety events for caregiver dashboard history.
