# Showcase Plan

## Demo Narrative
Demonstrate a complete smart wearable elderly care flow from sensing to caregiver response.

## Demo Segments
1. **Health Monitoring Live View**
   - Show HR, HRV, SpO₂, respiratory rate, temperature, and activity status.
2. **Gait + Fall Prevention**
   - Show real-time gait scoring and fall-risk class transitions.
   - Trigger preventive vibration alert scenario.
3. **Emergency Response**
   - Simulate automatic fall detection and SOS flow.
   - Show caregiver notification and GPS sharing.
4. **Cognitive/Wellness Layer**
   - Show routine adherence, hydration reminder, and medication tracking views.
5. **Predictive Healthcare**
   - Show long-term trend analysis and AI wellness recommendations.

## Must-Have Screens
- Real-time health dashboard
- Fall risk visualization
- Mobility trend report
- Sleep analytics
- Emergency event history
- Medication adherence tracking
- Caregiver portal notifications
