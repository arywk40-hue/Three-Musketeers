# Smart Wearable Elderly Care System

> Textile-based, non-invasive, continuous monitoring platform for independent living support.

**Team:** Pranay · Ariyan · Reman Dey  
**Institution:** IIT Mandi  
**Version:** 2.0 — Capability Blueprint  
**Date:** June 2026

---

## Overview

This project defines a smart wearable system that combines health monitoring, gait intelligence, fall-risk prevention, emergency response, caregiver visibility, and predictive healthcare analytics in one connected platform.

The system is designed for all-day comfort with lightweight textile sensors, washable modular electronics, BLE communication, mobile integration, and secure cloud-backed data management.
